- [ ] Run `RUNWAY_API_KEY=<key> node scripts/generate-runway-ads.js`
- [ ] Videos saved to `ad-videos/*.mp4` — review each one
- [ ] **Add `dog-anxiety-wrap` product to catalog** before running that ad
- [ ] Upload videos to Meta Ads Manager → Create → Video Ad

- [ ] Weekly cron will auto-generate new videos every Sunday 7am UTC once deployed
